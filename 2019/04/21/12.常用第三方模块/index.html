常用第三方模块

除了内建的模块，Python还有很多的第三方模块，在PyPI-the Python Package Index注册。

PIL

PIL：Python Imaging Library，已经是Python平台事实上的图像处理标准库了。PIL的功能非常强大，API却非常简单易用。PIL只支持到2.7，PIL上的兼容版本Pillow，加了很多新特性，可以支持到3.x。
