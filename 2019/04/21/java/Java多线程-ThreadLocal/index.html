总结

  1. 每个线程都有自己的局部变量，一个线程的本地变量对于其他线程是不可见的
  2. 独立于变量的初始化副本，ThreadLocal可以给一个初始值，而每个线程都会获得到这个初始化值的一个副本，这样能保证不同的线程都有一份拷贝
  3. 状态与某一个线程相关联，ThreadLocal不是用于解决共享变量的问题的，不是为了协调线程同步而存在的，而是为了方便每个线程处理自己的状态而引入的一个机制。
